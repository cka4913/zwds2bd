# 紫微斗數生日反推系統

一個以紫微斗數規則反推農曆生日（月份、日期、時辰）的前端 Web App。

## 線上 Demo

https://cka4913.github.io/zwds2bd/

## 目前功能

- 基本資料輸入：陰陽、性別、年、月、日、時、局數（可留空）
- 命宮位置輸入：4x4 十二宮宮格點選
- 紫微星位置輸入：4x4 十二宮宮格點選
- 缺資料補問：
  - 缺月份：只問 `右弼` 位置（4x4 宮格）
  - 缺時辰：只問 `文昌` 位置（4x4 宮格）
  - 缺局數/陰陽/性別：問第一大限與兄弟宮順逆
- 日期反推：根據紫微位置 + 局數 + 其他已知條件收斂候選日

## 流程

1. 輸入基本資料（至少需要年份）
2. 選命宮位置
3. 選紫微星位置
4. 系統按缺資料補問（右弼 / 文昌 / 大限順逆）
5. （可選）輸入三台八座
6. 顯示推算結果

## 核心規則（已對齊 iztro）

### 1) 右弼反推月份
- 口訣：戌上逆正右弼當
- 公式：`month = (xuIndex - youbiIndex + 12) % 12 + 1`

### 2) 文昌反推時辰
- 口訣：戌上逆時覓文昌
- 公式：`timeIndex = (xuIndex - wenchangIndex + 12) % 12`

### 3) 大限順逆
- 規則：陽男陰女順行；陰男陽女逆行
- 兄弟宮範圍對應：
  - 順行：`(局數+110) ~ (局數+119)`，例如水二局為 `112-121`
  - 逆行：`(局數+10) ~ (局數+19)`，例如水二局為 `12-21`

### 4) 紫微反推日期
- 使用「局數除日數」定位法
- 每個日子只取第一個可整除 offset 計算紫微索引（避免同一天多 offset 造成爆量）
- 局數未知時不展開全局數暴力枚舉
- 已知時辰時會再用命宮月時關係過濾

## UI 變更重點

- 命宮與紫微位置：已統一為 4x4 十二宮格
- Step 4 補問輸入（右弼、文昌）：已改為 4x4 十二宮格
- 右下固定為「亥」，左下固定為「寅」
